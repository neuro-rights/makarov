E23-A capacitor formed by two arbitrary conductors - MoM/BEM numerical solution

The MATLAB function E23.m is a stand-alone simulator of the capacitance between two conductors (a two-conductor problem) which utilizes the Method of Moments. Its general features include:

  • User friendly and intuitive programmatic GUI
  • The project I/O data may be saved in separate project file(s)
  • Implemented entirely in basic MATLAB (R2011a or later)
  • No MATLAB toolboxes are needed

The technical features include:

  • The first conducting object may be a plate, circle, brick, sphere, and a cylinder
  • The second conducting object may be a plate, circle, brick, sphere, and a cylinder
  • The objects may have arbitrary orientations and spacing, including large spacings
  • The objects may be assigned arbitrary values of the electric potential/voltage
  • The objects may not intersect
  • The program outputs capacitance value and charge/field/potential distributions on object surfaces and in surrounding space
  • The program optionally outputs electric field/potential in a plane and/or electric field/potential at a point
  • High-quality uniform and non-uniform triangular meshes generated by DISTMESH (Copyright 2004-2012 Per-Olof Persson) are used

Getting started

This section describes how the program executes using a default set of parameters.

Default parameters

  • When first opened, MATLAB program E23.m loads a default set of parameters.
  • This set models capacitance between a brick and a cylinder with the voltages of -/+0.5V, respectively
    • A green cross indicates a point in space where the electric field/potential will be calculated, along with the capacitance
    • A semi-transparent plane is a plane where the electric field/potential will be calculated and plotted, along with the capacitance

Run the default configuration

  • The program is executed using the Simulate menu.
  • The waitbars indicate CPU time necessary to run different numerical subroutines
  • The results are automatically updated after the simulation has been finished
  • The simulations cannot be interrupted

Observe the results

  • When the simulation is finished the figure updates automatically
  • Depending on user's input, the figure will include charge and/or field/potential distributions
  • All other numerical results are observed using Output data menu
  • If you want to change parameters of the observation plane and/or the observation point, use Output setup menu
  • Output setup menu re-computes the fields at new locations, but does not repeat the MoM simulation

Save the results/geometry configuration

  • The figure with charge distributions and/or field/potential distributions is saved using Save figure menu
  • All I/O numerical data (geometry, capacitance and field/potential at a point) is saved using Project menu
  • Charge distributions and field/potential distributions in a plane are not saved. They need to be recalculated

Detailed description of program menu

This section describes all program menu options.

Menu item - Project

  • This menu item opens and saves project files. Every project file is a *.mat file.

Menu item - Save figure

  • This menu item saves the figure with charge/field/potential distributions as a MATLAB figure or as a jpeg/png file

Menu item - First conductor

  • This menu item specifies electric potential/voltage applied to the first conductor(a plate, circle, brick, sphere, or a cylinder)
  • This menu item specifies all geometry parameters for the first conductor
  • You may extend/rotate/move the object as required
  • Once an object parameter is changed, the triangular mesh of the entire structure is recreated
  • Generation of larger high-quality meshes may require a significant time. Meshes with more than 30,000 triangles are not recommended
  • Option Triangle size allows us to select between uniform and non-uniform meshes (size ratio for boundary-to-center triangles may vary from 1 to about 0.02)
  • Highly-inhomogeneous meshes of high quality should have a large number of nodes; their assembly may require a significant amount of time
  • Option Triangle size is not applicable to a sphere, the side surface of a brick, and the side surface of a cylinder

Menu item - Second conductor

  • This menu item specifies electric potential/voltage applied to the second conductor(a plate, circle, brick, sphere, or a cylinder)
  • This menu item specifies all geometry parameters for the second conductor
  • You may extend/rotate/move the object as required
  • Once an object parameter is changed, the triangular mesh of the entire structure is recreated
  • Generation of larger high-quality meshes may require a significant time. Meshes with more than 30,000 triangles are not recommended
  • Option Triangle size allows us to select between uniform and non-uniform meshes (size ratio for boundary-to-center triangles may vary from 1 to about 0.02)
  • Highly-inhomogeneous meshes of high quality should have a large number of nodes; their assembly may require a significant amount of time
  • Option Triangle size is not applicable to a sphere, the side surface of a brick, and the side surface of a cylinder

Menu item - Output setup

  • This menu item specifies parameters of the observation plane and of the observation point
  • This menu item may be updated before or after the simulations
  • It also specifies the screen sizes of used markers (cones and a cross), and cone spacing
  • Cone size is directly proportional to the strength of the electric field; relative scale is used
  • When the observation plane is not selected, only the charge distribution is plotted
  • The charge distribution may be scaled using two parameters: relative positive bound and relative negative bound
  • All other numerical results are observed using Output data menu

Menu item - Modeling setup

  • This menu item specifies numerical parameters of the MoM algorithm. The default values are quite general
  • The first parameter is the dimensionless radius of the integration sphere, R
  • If a pair of triangles of "sizes" s1, s2 is separated by d, then R=d/sqrt(s1*s2)
  • For triangles spaced closer than R, the accurate numerical integration is performed
  • In this case, the inner potential integral is found analytically; the outer integral uses a Gaussian quadrature with a prescribed number of points (second menu item)
  • Otherwise, the center-point approximation is used while performing double integration

Menu item - Output data

  • This menu item lists calculated parameters including the capacitance between two objects
  • It also lists the E-field/potential at a point if required
  • The exact numbers of triangular patches are additionally listed as well as execution times

Button Simulate

  • This button starts MoM simulations
  • The simulations cannot be interrupted
  • The simulations include two parts: filling the MoM matrix and solving a system of equations
  • The first part is longer for small structures; the second part is longer for larger structures
  • Using structures with more than 30,000 triangular patches is not recommended

Button Reload

  • This button cleans up the previous solution and reloads the structure

Button Separate window

  • This button copies the GUI output figure into a separate figure window. You could zoom in, rotate, and modify the resulting figure as required

Button Exit

  • This button exits the module
