E74-Conducting object in an external AC magnetic field - MoM/BEM numerical solution

The MATLAB function E74.m is a stand-alone accurate simulator of eddy currents excited in a conducting object in an external (impressed or incident) AC magnetic field utilizing the Method of Moments. Its general features include:

  • User friendly and intuitive programmatic GUI
  • The project I/O data may be saved in separate project file(s)
  • Implemented entirely in basic MATLAB (R2011a or later)
  • No MATLAB toolboxes are needed

The technical features include:

  • A conducting object may be a sphere, brick, cylinder, or an arbitrary external object
  • The object may have arbitrary orientation
  • The object may have arbitrary conductivity
  • The external magnetic field may have arbitrary direction/magnitude/frequency
  • The module computes surface charge distributions, eddy current density distribution in a plane, and eddy current density at a point of interest
  • High-quality uniform and non-uniform triangular meshes generated by DISTMESH (Copyright 2004-2012 Per-Olof Persson) are used

For the further description of technical features, please see Sections 12.2 and 12.4 of Chapter 12 of the textbook
